# Codeora
코더라는 간단하지만 자주 사용하는 코드들을 저장하는 사이트입니다.

## 사용 스택
- `React`
- `TypeScript`
- `Styled-Components`
- `Firebase` (`Auth`, `Cloud`)
- `Recoil`, `Recoil-Persist` : 로그인한 사용자 정보 등 클라이언트 상태 관리에 사용
- `React-Query` : 사용자가 작성한 코드 글 조회 등 서버 상태 관리에 사용
- `React-Syntax-Highlighter` : 코드 작성 시 하이라이팅을 주기 위해 사용
